# MovieLens Analytics: групповой проект по анализу данных

## О проекте

Аналитический отчет на основе датасета MovieLens (small version) с использованием Python и Jupyter Notebook. Цель нашего проекта - не только провести анализ, но и создать модульную и тестируемую структуру кода, применив навыки парсинга, работы с данными, математическиго анализа и построения отчетов. 


### Моя роль в проекте: Тимлид 

Я отвечала за: 
- разработку класса Ratings (обработка и анализ данных о рейтингах фильмов)
- написание тестов для класса Ratings
- рефакторинг и оптимизацию (устранение проблемных мест в коде. В частности, в классе Links, что позволило ускорить парсинг кеша и загрузку данных)
- координацию команды (распределение задач, организация созвонов, контроль сроков)
- создание итогового отчета по классу Ratings + общая редактура отчета


### Технологии и инструменты

* Язык: Python 3
* Библиотеки: pandas, pytest, requests, BeautifulSoup, collections, datetime, re, json
* Инструменты: Jupyter Notebook, Git, парсинг IMDb

### Что реализовано

*Модуль movielens_analysis.py*

Содержит 4 основных класса:
- Ratings (моя зона ответственности): анализ распределения оценок по времени, расчет метрик (среднее, медиана, дисперсия), сегментация пользователей
- Movies: анализ фильмов, распределение по годам выпуска и жанрам
- Tags: работа с тегами - самые популярные, длинные, поиск по словам
- Links: парсинг данных с IMDb (бюджет, режиссер, кассовые сборы) с кешированием
- Tests: написание тестов для каждого метода (проверка типов, сортировки, корректности вычислений), всего 50+


*Тестирование*

- Тесты для каждого метода
- Проверка типов данных, корректности сортировки
- Запуск: `python -m pytest movielens_analysis.py -v`


*Аналитический отчет*

- Использование всех методов модуля
- История, основанная на данных
- Каждая ячейка содержит `%timeit` для оценки производительности
